# PHPでつくるフォーム

## Nonceって？

number used onceの略で使い切りの番号のこと

```
ナンス（ハッシュ）を作成し、フォームまたはアクションを介して送信し、
フォームデータを受け入れる前、またはアクションを実行する前に、ナンスを確認します。
```

## ハッシュって？

### ハッシュ化とは

ある値に決められた条件を与えて別の値にすること。

たとえば・・

```
番号 1341 があるとして、
条件が それぞれの値に3を足す だとする
すると、出力値は 4674 となる。

この出力値がハッシュ値である
```

こうすると、条件を知っていれば元の値がなんだったのかを解読することができる。
